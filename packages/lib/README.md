# `@aishow/lib`

This package contains shared utility functions, constants, and helper modules used across the entire AI-Net Turborepo workspace.

## Purpose

To provide a centralized location for pure functions, data formatters, and any domain-agnostic logic that can be shared between the frontend applications and other internal packages.

## Usage

```ts
import { formatData } from "@aishow/lib";

const formatted = formatData(rawInput);
```
