# AI-Net Marketing Site

This application covers the public-facing promotional material for the AI-Net platform. It provides high-level overviews of the platform’s capabilities and use cases for developers interested in Microsoft Extensions AI integrations.

## Overview

The Marketing Site is built with:
- **Next.js** for static and dynamic rendering.
- **Tailwind CSS** for responsive, mobile-first design.
- Animated with `framer-motion` and `gsap` (where applicable) for modern presentations.

## Getting Started

1. Navigate to this directory (or run from the root of the Turborepo workspace).
2. Start the development server:

```bash
npm run dev
```

Visit [http://localhost:3001](http://localhost:3001) to explore the site.
