# AI-Net Web App (Features Portal)

This Next.js application serves as the primary interactive portal for the AI-Net platform. It provides hands-on demonstrations and user interfaces for exploring various integrations with Microsoft's AI extensions.

## Features Currently Available

- **Code Generation**: AI-assisted code templating and generation.
- **Meeting Assistant**: Interactive tools for transcript parsing and summaries.
- **Learning Hub**: Conversational AI designed for tutoring and education.
- **Speech to Text**: End-to-end voice transcription interactions.
- **Text Generation**: General-purpose AI text rendering and composition tools.

## Getting Started

1. Navigate to this directory (or run from the root of the Turborepo workspace).
2. Start the development server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the portal.
